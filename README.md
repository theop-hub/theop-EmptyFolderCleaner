# theop-EmptyFolderCleaner
Finds empty folders and deletes them. This script only works on Windows.

# IMPORTANT INFO
This may clear out useful folders that are empty. In order to stop these folders from being deleted, the python file's list of exclusions must be edited. Add a new line and write the name of the root folder to be ignored in speech marks eg. "GitHub".

# How to use
- Download the latest release
- Extract it
- Run the .bat file
